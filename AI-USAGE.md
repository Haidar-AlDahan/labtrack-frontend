# AI Usage Report

## Overview
This project was developed with the assistance of AI tools to support learning, speed up development, and improve code quality. AI was used as a helper, not as a replacement for understanding or writing the project.

The main tool used was ChatGPT, Claude.

---

## How AI Was Used

### 1. Code Guidance and Debugging
AI was used to:
- Fix errors in React components and routing
- Debug issues related to navigation (e.g., sidebar routing, page linking)
- Resolve Git problems such as merging branches, rebasing, and conflicts
- Improve component structure (layouts like DashboardLayout, Sidebar, TopBar)

Example:
- Fixing broken navigation between pages like `/labs`, `/dashboard`, and `/grades`
- Handling merge conflicts when integrating feature branches into development

---

### 2. UI Design and Structuring
AI helped in:
- Suggesting layout structures for pages (Dashboard, Peer Review pages)
- Organizing components into reusable parts (SideBar, TopBar, Layouts)
- Improving user experience ideas like:
  - Peer review workflow
  - Dashboard structure
  - Page organization

---

### 3. Feature Implementation
AI assisted in implementing features such as:
- Peer review pages (Assigned reviews, Received reviews)
- Dashboard layout integration across pages
- Logout interaction from UI elements
- Page routing structure using React Router

However, all logic and final decisions were reviewed and adjusted manually.

---

### 4. Documentation Writing
AI was used to review :
- README file
- Project setup instructions
- This AI usage report

To make sure they were written clearly and accuretly.

---

## What AI Was NOT Used For

- AI did not complete the entire project automatically
- AI did not replace understanding of concepts taught in the course
- Final code structure, integration, and testing were done manually
- Decisions about project design and features were made by the team

---

## Verification and Learning

To ensure proper learning:
- All AI-generated code was reviewed and understood before use
- Code was tested and modified manually
- Concepts like routing, component structure, and state handling were learned and applied independently

---

## Conclusion

AI was used as a support tool to:
- Speed up debugging
- Improve code structure
- Assist with ideas and documentation

The project reflects the student's own understanding, with AI serving only as a guide.
